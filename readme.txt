== Description ==

<h3>Display About Me Section easily on your WordPress sidebar widgets</h3>

Easy Profile Widget is used to display a user profile section with avatar and custom read more link. It can be used in any widget area, such as a sidebar or a footer.

<h3>Easy Setup</h3>
After plugin activation 'Easy Profile' widget will be available under <strong>Appearance > Widgets</strong>, just drag and drop to any desired widget area.

<h3>SiteOrigin Page Builder Compatible</h3>
Widgetized plugins cannot be completed without SiteOrigin compatibility! If you are using SiteOrigin Page Builder and you want to easily add About Me or Profile Section on your content, Easy Profile Widget is what you're looking for! Go ahead and try it :)

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. That's it :)
